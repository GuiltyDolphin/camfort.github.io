Download
The source code for CamFort is hosted on GitHub.

Binary releases for Windows, Mac OS, and Linux are available. Current binary release: v0.900
Project
CamFort is a joint project between the University of Cambridge and University of Kent supported by an EPSRC grant.

Find out about our team and see a list of publications.
Mailing list
For help and announcments you can sign-up to our mailing list.
Events
An introduction to CamFort and training material is provided as part of the Fortran Modernization Workshops run by the Numerical Algorithms Group (nag) in the UK and Europe.

We host an annual event on Testing and Verification in Computational Science. Previous versions occured in 2016 and 2017.
